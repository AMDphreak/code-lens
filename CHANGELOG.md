# Changelog

## [0.1.0] - 2026-05-28

### Added

- Portable JSON5 spec (`spec/`) for examples, themes, and UI behavior.
- `@examplens/core` — parse, validate, variable slot detection, theme resolution.
- `@examplens/vanilla` — `<lens-code-block>` with hover preview, glass tab indicator, swipe, morphing diff tokens.
- Six color schemes: tropical, earth, earthenware, patina, fruits, desert — each with distinct panel color per lens.
- Demo site (Vite) for GitHub Pages.
- Specification, AI reproduction spec, and implementation registry.
